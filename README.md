# Dotfiles

Repo dotfiles untuk konfigurasi lingkungan kerja.

Folder yang ada:

- `hypr/` — konfigurasi untuk Hyprland (Wayland compositor).
- `waybar/` — konfigurasi untuk Waybar (status bar untuk Wayland).

Cara cepat menggunakan:

1. Clone atau letakkan folder ini di `~/.config` atau lokasi yang sesuai.
2. Sesuaikan file konfigurasi di masing-masing folder dengan preferensi pribadi.

Contoh: jika ingin menggunakan konfigurasi Hyprland:

```bash
# pindah ke direktori konfigurasi
mkdir -p ~/.config
cp -r hypr ~/.config/
cp -r waybar ~/.config/
```
